# Biblioteca Led (Arduino)

Biblioteca desenvolvida para controle não bloqueante de um LED utilizando Arduino.

Permite ligar, desligar, controlar tempo e criar efeitos de piscada sem o uso de delay(), garantindo melhor desempenho e organização do código.

---

## 📌 Funcionalidades

- Ligar o LED continuamente
- Desligar o LED
- Ligar o LED por tempo determinado
- Piscar continuamente (frequência padrão ou personalizada)
- Piscar por uma quantidade definida de vezes
- Alternar o estado do LED
